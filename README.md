# LZT Market Hub

Десктопная программа для работы с [Lolzteam Market](https://lzt.market) на Windows и macOS. Один инструмент вместо десятка вкладок браузера, ботов и таблиц: массовая выгрузка аккаунтов, фоновый мониторинг новых лотов, автопокупка, журнал перепродаж, аналитика по конкурентам и управление финансами.

Проект написан для [конкурса опенсорс-инструментов Lolzteam](https://lolz.live/threads/10101986/) и распространяется бесплатно с открытым исходным кодом (лицензия MIT).

## Для кого это

- **Продавцам** — заливать аккаунты пачками из CSV, следить за ценами, автоматически поднимать и переоценивать лоты.
- **Перекупам** — вести журнал сделок с расчётом прибыли, быстро перепродавать купленное, передавать аккаунты другим пользователям.
- **Покупателям** — ловить выгодные лоты раньше других: фоновый мониторинг проверяет маркет каждые несколько секунд и умеет покупать сам по заданным правилам.
- **Всем остальным** — смотреть статистику своего профиля, историю платежей, переводить и выводить деньги, не открывая браузер.

Если вы не программист — ничего страшного. Программа ставится как обычное приложение, внутри всё на русском (есть и английский), а при первом запуске мастер настройки проведёт по шагам.

## Что умеет

**Дашборд.** Сводка по аккаунту: баланс, холд, активные лоты, последние события. Графики продаж и динамика цен.

**Центр выгрузки.** Массовая загрузка аккаунтов на продажу из CSV-файла через официальный fast-sell API. Понимает форматы login:password и почтовые данные, умеет прикреплять maFile для Steam, проверяет дубликаты до выгрузки, подставляет заголовки и цены по шаблону. После выгрузки следит за лотами: что продалось, что висит, где менялась цена.

**Монитор покупателя.** Работает в фоне даже при свёрнутом окне. Вы сохраняете фильтр поиска (категория, цена, любые параметры маркета), и программа опрашивает маркет с заданным интервалом. Новый лот — мгновенное уведомление на рабочий стол, в Telegram или вебхуком. Есть правила: например «если цена ниже N и продавец не в чёрном списке — купить автоматически». У автопокупки есть предохранители: дневной лимит, потолок цены, сверка с рекомендованной ценой маркета.

**Стол перекупа.** Журнал сделок: что купили, за сколько, за сколько продали, какая вышла прибыль. Считает ROI, строит графики по неделям и месяцам. Отсюда же можно перепродать аккаунт, отменить гарантию или передать лот другому пользователю.

**Автоматизация.** Автоподнятие лотов по расписанию, массовые действия над десятками лотов сразу (поднять, открыть, закрыть, сменить цену, навесить тег) — через batch API, чтобы не упираться в лимиты. Автопереоценка залежавшихся лотов: висит дольше N дней — цена снижается на заданный процент.

**Аналитика.** Синхронизация профиля, графики продаж и доходов. Отслеживание конкурентов: добавляете продавца — программа собирает его лоты, сопоставляет с вашими и показывает, где вас перебивают по цене.

**Финансы.** Балансы во всех валютах, обмен между ними, переводы другим пользователям (с учётом комиссии и холда), вывод на кошельки, история платежей, инвойсы и автоплатежи.

**Инструменты.** Уведомления в Telegram через своего бота, настройка IMAP для почтовых доменов, менеджер прокси, проверка на дубликаты, быстрый поиск по маркету, экспорт чего угодно в CSV.

**Аккаунт и настройки.** Страница профиля открывается кликом по нику в боковой панели. В настройках — токен, язык, валюта, включение и отключение модулей, секретный ответ для переводов.

## Установка

### Готовая сборка

Скачайте установщик из Releases: `.dmg` для macOS или `.exe` для Windows. Установите и запустите.

### Из исходников

Понадобится [Node.js](https://nodejs.org) версии 20 или новее.

```bash
git clone https://github.com/flatherflether-dot/LZT-Market-HUB.git
cd lzt-market-hub
npm install
npm run dev
```

Собрать установщик самому:

```bash
npm run dist:mac   # .dmg для macOS
npm run dist:win   # установщик для Windows
```

Готовые файлы появятся в папке `release`.

## Первый запуск

1. Получите API-токен: [lolz.live/account/api](https://lolz.live/account/api) → создайте приложение с правами **market** (для истории платежей и переводов добавьте **payment**).
2. Запустите программу — мастер настройки попросит вставить токен.
3. По желанию укажите секретный ответ (нужен только для передачи аккаунтов и переводов, хранится локально).
4. Не хочется сразу подключать свой аккаунт? Нажмите «Загрузить демо-данные» — программа наполнится тестовыми лотами и сделками, можно всё пощупать без токена.

Для работы с Market API на аккаунте должно быть 200 симпатий либо купленная подписка — это ограничение самого маркета, не программы.

## Безопасность

- Токен хранится только на вашем компьютере, в локальной базе данных программы. Никаких сторонних серверов, статистики и телеметрии нет.
- Все запросы идут напрямую на официальные адреса `prod-api.lzt.market` и `prod-api.lzt.live` и больше никуда. Это легко проверить по исходному коду — он весь открыт.
- База данных (сделки, история выгрузок, фильтры) автоматически бэкапится раз в сутки, хранятся последние 7 копий.

## Как программа обращается с лимитами API

Маркет ограничивает частоту запросов, и программа соблюдает эти лимиты сама — вам не нужно ни о чём думать:

- встроенный ограничитель повторяет официальные корзины лимитов (GET 300/мин, изменяющие запросы 30/мин, поиск 120/мин, письма 5/мин, проверка аккаунтов 300/мин, редактирование и подтверждение покупки 1000/мин, batch 20/мин);
- читает заголовки `X-RateLimit-*` и данные `system_info.rate_limit` из ответов и подстраивается на лету;
- при ответе 429 или ошибке `retry_request` повторяет запрос автоматически, как рекомендует документация.

Документация API: [lzt-market.readme.io](https://lzt-market.readme.io/reference/information).

## Технологии

Electron, React, TypeScript, SQLite (better-sqlite3), Zustand, Recharts. Каждый модуль — отдельная папка в `src/modules` со своим манифестом, страницей и переводами; модули можно отключать в настройках.
